Installation & Setup
1
Install dependencies
Requires Node.js ≥ 18 and ffmpeg on PATH
# Install ffmpeg (required for audio/video)
brew install ffmpeg           # macOS
sudo apt install ffmpeg       # Ubuntu/Debian
winget install Gyan.FFmpeg    # Windows

# Install Node deps
npm install
2
Build the server
npm run build
3
Add to Claude Desktop config
Edit ~/Library/Application Support/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "file-converter": {
      "command": "node",
      "args": ["/path/to/file-converter-mcp/dist/index.js"]
    }
  }
}
4
Restart Claude Desktop — start converting!
Ask Claude: "Convert my beat.mp3 to WAV at 48kHz" or "Generate an icon set from my logo.png"
Example Prompts
# Audio
"Convert /music/beat.mp3 to WAV at 48000Hz stereo"
"Batch convert all MP3s in /stems/ to FLAC"

# Video
"Convert broll.mov to WebM, trim to first 30 seconds"
"Extract audio from interview.mp4 as a WAV file"

# Image
"Convert logo.png to WebP at quality 90, 800px wide"
"Generate a full icon set from app-icon.png"

# Documents
"Convert README.md to JSON for my API response"
"Transform data.csv to JSON, then to Markdown table"
